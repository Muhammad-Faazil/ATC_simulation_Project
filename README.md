# Finite Automata Simulator

This project is a web-based simulator for deterministic (DFA) and non-deterministic (NFA) finite automata. It provides an interactive way to define, visualize, and simulate finite automata.

## Features

*   **Automaton Definition:** Easily define an automaton by specifying its states, alphabet, start state, accept states, and transition rules.
*   **Interactive Visualization:** The application uses the `vis.js` library to generate an interactive graph of the automaton.
*   **Simulation:** Test your automata with input strings to see if they are accepted or rejected.
*   **Step-by-Step Simulation:** Walk through the simulation step-by-step to understand how the automaton processes the input.
*   **Sample Automata:** Load pre-built sample automata to explore different types of DFAs and NFAs.
*   **Import/Export:** Save your automaton to a JSON file and import it back later.

## How to Use

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Muhammad-Faazil/ATC_simulation_Project.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd FiniteAutomataSimulator
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Start the server:**
    ```bash
    npm start
    ```
5.  Open your web browser and go to `http://localhost:3000`.
